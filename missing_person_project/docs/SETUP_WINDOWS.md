## ✅ System Requirements

- **Python**: 3.9 or 3.10 (your system has 3.10.0 ✓)

Before installing, ensure you have:

- [ ] Python 3.9 or 3.10 installed

- [ ] Internet connection (for package downloads)
- [ ] ~500 MB disk space free

## 🚀 Step 1: Prepare Your Data

### 1.1 Reference Image

Place a clear photo of the missing person in:
```
missing_person_project/data/input/missing.jpg
```

**Requirements:**
- Clear, frontal face
- Good lighting (not too bright/dark)
- At least 100x100 pixels
- JPG format (or PNG, BMP)

### 1.2 Video Files

Place your CCTV video files in:
```
missing_person_project/data/input/cam1.mp4
missing_person_project/data/input/cam2.mp4
```

**Supported formats:**
- MP4, AVI, MOV, MKV, WMV
- Any OpenCV-supported format

**If you have different video paths:**
- Edit `src/config.py`
- Change `camera_sources` variable
- Update paths to your actual video locations

## 🔍 Troubleshooting

### Problem: "No module named 'face_recognition'"

**Solution:**
```powershell
# Reinstall specifically
pip install face-recognition==1.3.0 --no-cache-dir

# Verify
python -c "import face_recognition; print('OK')"
```

### Problem: "Python 3.11 required"

**Solution:**
Your Python 3.10 is fine. The error is from numpy version mismatch.
```powershell
# Reinstall requirements
pip uninstall numpy opencv-python -y
pip install -r requirements.txt --upgrade
```

### Problem: "Can't find video file"

**Solution:**
1. Verify file exists: `dir data/input/cam*.mp4`
2. Check spelling in `src/config.py`
3. Use full path if needed:
```python
camera_sources: tuple[Path, Path] = (
    Path("C:/path/to/cam1.mp4"),
    Path("C:/path/to/cam2.mp4"),
)
```

### Problem: "No detections recorded"

**Solutions (in order):**
1. Run diagnostics: `python scripts/diagnose_system.py`
2. Check reference image is clear
3. Check video quality (faces visible?)
4. Lower confidence threshold to 0.35
5. Switch model to CNN: `model_preference = "cnn"`

### Problem: "Slow Processing / High CPU"

**Solutions:**
1. Skip more frames: `initial_frame_skip = 3`
2. Use HOG detector: `model_preference = "hog"`
3. Disable windows: `show_windows = False`
4. Reduce video resolution (optional)

### Problem: "Access Denied" or Permission Errors

**Solution:**
```powershell
# Run as administrator
Start-Process powershell -argumentlist "cd 'path'; python run.py" -Verb RunAs
```
